Este projeto foi desenvolvido para praticar lógica de programação.  
Ele simula um ambiente de venda de ingressos, onde o usuário pode visualizar a quantidade disponível e realizar compras, que reduzem o total de ingressos restantes.

---

## Funcionalidades

- Exibe o total inicial de ingressos disponíveis  
- Permite ao usuário escolher quantos ingressos deseja comprar  
- Atualiza automaticamente o número de ingressos restantes  
- Impede a compra caso a quantidade desejada ultrapasse o total disponível  
- Exibe mensagens informando o sucesso da compra ou a indisponibilidade

---

## Tecnologias utilizadas

- Linguagem: JavaScript
- Módulos nativos: input/output e controle de fluxo  
- Interface: linha de comando (modo texto)

---

## Objetivo de aprendizado

- Praticar estruturas condicionais e de repetição  
- Manipular variáveis e valores numéricos  
- Simular um sistema de controle de estoque simples  
- Trabalhar com interação básica com o usuário
